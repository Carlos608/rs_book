Sensoriamento Remoto da Amazônia: Do Píxel ao Dado (2026)
Por: Carlos Souza Jr.
Este repositório contém os códigos, notebooks e exemplos práticos que acompanham o livro "Sensoriamento Remoto da Amazônia". O objetivo aqui é transformar a teoria física em prática computacional utilizando o Google Earth Engine (GEE) e Python.
